# Utilities Directory

This directory will contain utility functions for data processing, evaluation, and visualization.

## Structure

```
utils/
├── __init__.py
├── data_loader.py         # Data loading and preprocessing
├── metrics.py             # Evaluation metrics
└── visualization.py       # Visualization tools
```

## Utilities to be Released

### Data Loading
- PEMS dataset loaders (PEMS03, PEMS04, PEMS07, PEMS08)
- Data normalization and scaling
- Missing value handling
- Train/validation/test splitting

### Evaluation Metrics
- MAE (Mean Absolute Error)
- RMSE (Root Mean Square Error)
- MAPE (Mean Absolute Percentage Error)
- Custom metrics for IoT sensor data quality

### Visualization Tools
- Traffic flow predictions plots
- Attention weight visualization
- Dynamic segmentation boundary plots
- Model performance analysis

**Note**: Complete utility implementations will be released upon paper acceptance.
