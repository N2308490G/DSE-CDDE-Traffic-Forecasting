# Baseline Models

This directory will contain implementations of baseline models for comparison.

## Baseline Models to be Included

### Traditional Methods
- ARIMA
- Vector Autoregression (VAR)

### Deep Learning Methods
- LSTM
- GRU
- Seq2Seq with Attention

### Graph Neural Network Methods
- DCRNN (Diffusion Convolutional Recurrent Neural Network)
- STGCN (Spatio-Temporal Graph Convolutional Network)
- Graph WaveNet
- ASTGCN (Attention-based Spatial-Temporal Graph Convolutional Network)
- MTGNN (Multi-hop Temporal Graph Neural Network)

### Recent State-of-the-Art Methods
- STAEformer
- D2STGNN
- STID
- PDFormer
